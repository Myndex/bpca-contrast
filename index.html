How to use

As you type, the contrast ratio indicated will update. Hover over the circle to get more detailed information. With alpha on backgrounds, the contrast value will indicate an error margin.

This 18px serif text visually demonstrates
how readable this color combination is,
for normal, italic, bold, or bold italic
text of various sizes and font styles.

Hint: Press the up and down keyboard arrows while over a number inside a functional color notation. Watch it increment/decrement. Try with the Shift or Alt key too!

This tool is largely based on the popular WCAG_2 contrast tool by Lea Verou.

It is modified for "Bridge PCA" by Myndex, the reason we chose this as the basis was to provide a familiar tool to demonstrate Bridge PCA.
